# QuickWheel 隐私政策

**生效日期：** 2026年3月5日

## 概述

QuickWheel（"我们"、"我们的"或"本应用"）尊重用户的隐私。本隐私政策说明我们如何收集、使用和保护您的信息。

## 信息收集

### 我们不收集的信息
- 个人身份信息
- 使用数据
- 分析数据
- 网络数据

### 本地存储的数据
本应用仅在您的本地计算机上存储以下数据：

1. **配置文件** (`config.json`)
   - 快捷项设置（名称、路径、图标等）
   - 热键配置
   - 应用设置（颜色、大小等）
   - 存储位置：`%AppData%\QuickWheel\config.json`

2. **注册表项**
   - 开机自启动设置
   - 位置：`HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`

## 数据使用

所有数据仅用于：
- 保存您的应用设置
- 实现应用功能
- 提供个性化体验

## 数据共享

我们不会：
- 收集或传输任何数据到远程服务器
- 与第三方共享任何数据
- 出售任何用户数据

## 数据安全

- 所有数据仅存储在您的本地设备上
- 配置文件使用标准 JSON 格式，您可以随时查看和修改
- 应用不需要网络连接即可运行

## 用户权利

您拥有以下权利：
- **访问**：随时查看配置文件内容
- **修改**：通过应用界面或手动编辑配置文件
- **删除**：卸载应用将删除所有本地数据
- **导出**：支持导出配置为 JSON 文件
- **导入**：支持从 JSON 文件导入配置

## 权限说明

本应用需要以下系统权限：

1. **全局热键**
   - 用于监听热键组合（默认 Ctrl+Alt+Q）
   - 仅在应用运行时有效

2. **文件系统访问**
   - 读取配置文件
   - 执行快捷项中的文件/程序

3. **开机自启**（可选）
   - 仅在您启用此功能时使用

## 第三方服务

本应用不使用任何第三方服务、SDK 或分析工具。

## 儿童隐私

本应用不针对13岁以下儿童，也不会故意收集儿童的个人信息。

## 政策更新

我们可能会更新本隐私政策。更新后的政策将在应用内和发布页面显示。

使用本应用即表示您同意本隐私政策的条款。
