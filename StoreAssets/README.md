# 微软商店发布资源

## 应用截图要求

需要准备以下尺寸的截图（.png 格式）：

### 桌面截图
- **1366 x 768** (16:9) - 推荐
- **1920 x 1080** (16:9) - 推荐
- **2560 x 1440** (16:9)

建议截图内容：
1. 轮盘菜单呼出状态
2. 设置窗口 - 快捷项管理
3. 设置窗口 - 热键与轮盘设置
4. 系统托盘菜单

## 商店展示图

### 特色图片 (Hero Image)
- **3840 x 2160** 或 **2560 x 1440**
- 用于商店首页展示

### 宽屏展示图
- **1920 x 1080**
- 用于商店详情页顶部

### 详细描述
```
QuickWheel 是一款高效的快捷操作工具，帮助您通过简单的手势快速执行常用任务。

主要功能：
• 圆形轮盘菜单 - 按住热键呼出，鼠标移动选择，松手执行
• 多种操作类型 - 打开文件、文件夹、网址，执行命令，发送快捷键
• 自定义配置 - 支持 Emoji 图标、程序图标，可调整颜色和样式
• 快捷项管理 - 添加、删除、拖拽排序，导入导出配置
• 系统托盘 - 常驻后台，随时可用
• 开机自启 - 可选开机自动启动
• 模板变量 - 支持剪贴板内容、选中文本、日期时间等变量

使用场景：
• 快速打开常用软件、文件夹
• 一键访问常用网站
• 执行常用命令
• 发送快捷键组合
• 提升日常工作效率

支持 Windows 10/11 系统。
```
